Jillegal-compiler

Java Generic and Multi Language Supported In Memory Compiler

Download this project as a .zip file Download this project as a tar.gz file

What is Jillegal-Compiler?

Jillegal-Compiler is a JMV based on the fly code compilating and class generation library.

What features does Jillegal-Compiler have?

  • Only the fly in memory Java code compilation and class generation.

  • Only the fly in memory Groovy code compilation and class generation.

  • Flexible design to allow for supporting new code types such as Ruby, Phyton, ....

Install

In your pom.xml, you must add repository and dependency for Jillegal-Compiler. You can change jillegal.compiler.version to any existing jillegal-compiler library version.

...
<properties>
    ...
    <jillegal.compiler.version>1.0.0-RELEASE</jillegal.compiler.version>
    ...
</properties>
...
<dependencies>
    ...
    <dependency>
        <groupId>tr.com.serkanozal</groupId>
        <artifactId>jillegal-compiler</artifactId>
        <version>${jillegal.compiler.version}</version>
    </dependency>
    ...
</dependencies>
...
<repositories>
    ...
    <repository>
        <id>serkanozal-maven-repository</id>
        <url>https://github.com/serkan-ozal/maven-repository/raw/master/</url>
    </repository>
    ...
</repositories>
...

Jillegal-Compiler with simple examples

You can access all demo codes from here: jillegal-compiler-demo

Java Class Compiler Example

ClassCompilerFactory classCompilerFactory = new DefaultClassCompilerFactory();

String code = 
    "package tr.com.serkanozal.jillegal.compiler;"+ "\n" +
    "\n" +
    "public class SampleJavaClass {" + "\n" +
    "\n" +
    "\t" + "public String toString() {" + "\n" +
    "\t" + "\t" + "return \"I am SampleJavaClass\";" + "\n" +
    "\t" +  "}" + "\n" +
    "\n" +
    "}";

Class<?> compiledClass = classCompilerFactory.getClassCompiler(DefaultCodeType.JAVA).compile(code);
System.out.println("In Memory Compiled Java Class: " + compiledClass.getName());
Object obj = compiledClass.newInstance();
System.out.println(obj.toString());

Output:

In Memory Compiled Java Class: tr.com.serkanozal.jillegal.compiler.SampleJavaClass
I am SampleGroovyClass

Groovy Class Compiler Example

ClassCompilerFactory classCompilerFactory = new DefaultClassCompilerFactory();

String code = 
    "package tr.com.serkanozal.jillegal.compiler;"+ "\n" +
    "\n" +
    "public class SampleGroovyClass {" + "\n" +
    "\n" +
    "\t" + "public String toString() {" + "\n" +
    "\t" + "\t" + "def str = \"I am SampleGroovyClass\";" + "\n" +
    "\t" + "\t" + "return str;" + "\n" +
    "\t" +  "}" + "\n" +
    "\n" +
    "}";

Class<?> compiledClass = classCompilerFactory.getClassCompiler(DefaultCodeType.GROOVY).compile(code);
System.out.println("In Memory Compiled Groovy Class: " + compiledClass.getName());
Object obj = compiledClass.newInstance();
System.out.println(obj.toString());

Output:

In Memory Compiled Groovy Class: tr.com.serkanozal.jillegal.compiler.SampleGroovyClass
I am SampleGroovyClass
